# Positioning Map (FICTIONAL EXAMPLE)

> Illustrative only.

## Axes

- X-axis: horizontal clinical scribe to PT-specialist workflow agent
- Y-axis: lightweight dictation aid to embedded documentation and coding
  system

## Observations

- Horizontal scribes occupy the lower-left to middle: broad clinical
  transcription with limited PT-specific workflow depth.
- EHR-first PT tools occupy the upper-middle: embedded workflow, but
  documentation automation is a secondary feature.
- Manual EHR macros are embedded but low automation.
- The open gap is a PT-specialist scribe-first product that handles notes,
  CPT coding, and exercise prescription as one workflow.

## Strategic Gap

Position as **PT-native documentation automation**, not "another AI
scribe." The wedge must prove that vertical depth produces better notes
and fewer claim-review headaches than horizontal transcription.
